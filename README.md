# Smart Calculator

A powerful and user-friendly Smart Calculator with enhanced features designed to provide a clear and intuitive experience for users. 

## Features

- **AI Chatbot**: An integrated AI chat assistant that solves problems step-by-step. You can upload questions or search for problems, and the AI will provide a clear step-by-step solution. For example, even for a problem like `2*5%4(2+7)`, the AI will show how the BODMAS rule is implemented and solve each step clearly, allowing users to understand the logic (like how a result such as `56` is derived) without any confusion.
- **Color Themes**: Customizable color themes (color-changeable) for an improved and personalized user experience.
- **Calculation History**: Automatically stores previous calculations in the history so you can easily review past operations.
